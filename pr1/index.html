Разработка простой веб-страницы с использованием HTML5

World Wide Web (www, web, рус.: веб, Всемирная Паутина) — распределенная информационная
система, предоставляющая доступ к гипертекстовым документам по протоколу HTTP(HyperText
Transfer Protocol). WWW - сетевая технология, построенная на клиент-серверной архитектуре и
использующая инфраструктуру Интернет для взаимодействия между сервером и клиентом (рис. 1).
